## üöÄ Overview

CityQuest gamifies real-world running by turning your city into a fog-covered map waiting to be explored. Players unlock districts, collect supply drops, and compete on global leaderboards by physically running in the real world.

### Key Features
*   **Fog of War**: The world is covered in fog. Run to clear it and reveal the map.
*   **District Control**: Conquer city districts by exploring their area.
*   **Supply Drops**: Daily location-based loot boxes that encourage route variety.
*   **Competitive Leagues**: Weekly promotions/relegations based on activity.
*   **Real-time Multiplayer**: See other runners' progress and stats.
